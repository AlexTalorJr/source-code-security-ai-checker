**Create `src/scanner/core/orchestrator.py`** with these functions:

1. `def deduplicate_findings(findings: list[FindingSchema]) -> list[FindingSchema]`:
   - Group findings by `fingerprint` field
   - For duplicates, keep the finding with highest severity (Severity is IntEnum, higher value = more severe)
   - If same severity, keep first encountered
   - Return deduplicated list preserving insertion order (use dict[str, FindingSchema])
   - This is a pure function, no async needed

2. `async def _run_adapter(adapter: ScannerAdapter, target_path: str, timeout: int, extra_args: list[str] | None) -> tuple[str, list[FindingSchema] | Exception]`:
   - Wrapper that calls `adapter.run(target_path, timeout, extra_args)`
   - Returns `(adapter.tool_name, findings_list)` on success
   - Returns `(adapter.tool_name, exception)` on any exception (catches all exceptions)
   - This wrapper exists so `asyncio.gather` gets per-adapter error isolation

3. `async def run_scan(settings: ScannerSettings, target_path: str | None = None, repo_url: str | None = None, branch: str | None = None) -> ScanResultSchema`:
   - Validates: either `target_path` or `(repo_url + branch)` must be provided, not both, not neither. Raise `ValueError` if invalid.
   - Creates `ScanResultSchema` with `status="running"`, `started_at=datetime.utcnow()`
   - If `repo_url` provided:
     - Determine shallow: `shallow = True` (default for all tools)
     - Check if gitleaks is enabled -- if so, use `shallow=False` (Gitleaks needs full history)
     - Call `clone_repo(repo_url, branch, shallow=shallow, git_token=settings.git_token)`
     - Set `target_path` to the cloned directory
     - Wrap entire scan in try/finally that calls `cleanup_clone(clone_path)`
   - Build list of enabled adapters: iterate `ALL_ADAPTERS`, check `getattr(settings.scanners, adapter_class.tool_name).enabled`
   - For each enabled adapter, get its config: `tool_config = getattr(settings.scanners, adapter.tool_name)`
   - Create async tasks: `[_run_adapter(adapter(), target_path, tool_config.timeout, tool_config.extra_args or None) for ...]`
   - Run all with `asyncio.gather(*tasks)` (NO return_exceptions -- _run_adapter handles exceptions internally)
   - Collect results: separate successful findings and warnings
   - For each result: if isinstance(result, Exception), add to warnings list as string `f"{tool_name}: {str(exception)}"`; else extend all_findings
   - Collect tool versions: for each successful adapter, try `await adapter.get_version()` (catch exceptions, use "unknown")
   - Call `deduplicate_findings(all_findings)` on collected findings
   - Count findings by severity:
     ```python
     counts = {s: 0 for s in Severity}
     for f in deduped_findings:
         counts[f.severity] += 1
     ```
   - Determine quality gate: `gate_passed = (counts[Severity.CRITICAL] + counts[Severity.HIGH]) == 0`
   - Fill ScanResultSchema:
     - `status="completed"`, `completed_at=datetime.utcnow()`
     - `duration_seconds=(completed_at - started_at).total_seconds()`
     - `total_findings=len(deduped_findings)`
     - `critical_count`, `high_count`, etc. from counts
     - `gate_passed=gate_passed`
     - `tool_versions=tool_versions_dict`
     - `error_message="\n".join(warnings)` if warnings, else None
     - `target_path`, `repo_url`, `branch` from args
   - Persist to SQLite: create engine from `settings.db_path`, ensure tables exist via `Base.metadata.create_all`, create session, insert `ScanResult` ORM object with all fields, then insert `Finding` ORM objects for each deduplicated finding with `scan_id` set
   - Return ScanResultSchema with `id` populated from DB insert

4. `def format_summary_table(scan_result: ScanResultSchema, warnings: list[str]) -> str`:
   - Returns a formatted string (NOT using rich -- just plain text for now, rich used in CLI layer)
   - Includes: table of (tool, findings_count, status), total findings, severity breakdown, gate result, duration
   - If warnings non-empty, prepend WARNING banner lines
